DICOM
=====

This normative section contains a description of the DICOM tags which are
necessary to fully describe orthodontic views (photographs).

This part was compiled using the various values allowed in the DICOM CID
(Content IDs) tables, removing those values which are not relevant to the
orthodontic domain.
